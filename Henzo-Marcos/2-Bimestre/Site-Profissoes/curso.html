Sobre o curso

O curso de Desenvolvimento de Software prepara os alunos para criar sistemas, aplicativos, sites e outras tecnologias digitais.

Durante o curso, os estudantes aprendem programação, banco de dados, lógica computacional e desenvolvimento web.

Duração

A duração média varia entre 2 e 4 anos, dependendo da faculdade e do tipo de graduação.

Principais matérias

  • Lógica de Programação
  • Banco de Dados
  • Desenvolvimento Web
  • Engenharia de Software
  • Redes de Computadores

Faculdades que oferecem o curso

  • USP
  • UNICAMP
  • FIAP
  • PUC
  • SENAI

Modalidades

O curso pode ser encontrado nas modalidades presencial, semipresencial e EAD.
